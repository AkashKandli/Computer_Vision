To install all necessary packages run 
pip install -r requirements.txt

To run customized hough transform on image with circles (many_circles.png) run 
python still.py 


To run customized hough transform on video (ball_video.mp4) run 
python main.py 


All generated images are in the generated_images directory. The env directory contains the virtual environment I am using. You don't need to run these python files in a virtual environment, although it might make things more convenient.
